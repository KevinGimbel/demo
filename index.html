imageSwitcher

A 640 byte JavaScript Function that turns a set of images into a interactive set of images.

Usage:

        
        imageSwitcher({
          previewImage: "[data-js='previewImage']", // preview image 
          allImages: ".thumbnails > img", // all the images for the array
          event: "mouseenter" // the event to trigger the function
        });

You can see the source code here on Github.

dull-js

In honor of the movie "The Shining" from 1980.

Usage:

        
          dull("name","gender","verb");

          dull("Kevin","male","sleep");
          # Output "All work and no sleep makes Kevin a dull boy."

          dull("Jessica","female","fun");
          # Output "All work and no fun makes Jessica a dull girl."

Made for bullgitYou can see the source code here on Github.

fixed-elements

A tiny JavaScript snippet to fix elements to a specific possition as soon as the pages offset (window.scrollY) reachs a specific value.

Usage:
