Just another Photoset

A 640 byte JavaScript Function that turns a set of images into a interactive set of images.

Takes 3 Parameters:

  • previewImage: An <img> Tag that holds the big image
  • allImages: the container of all images, like .thumbnails > img
  • an event like mouseenter, mouseout or click
      
  imageSwitcher({
    previewImage: "[data-js='previewImage']", // preview image 
    allImages: ".thumbnails > img", // all the images for the array
    event: "mouseenter" // the event to trigger the function
  });

Images via LoremPixel.com, Demo with click Event on CodePen.
